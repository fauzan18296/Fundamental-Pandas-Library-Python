üß† 1. Apa itu Pandas dan Kenapa Penting?

üìç Pandas adalah library Python yang digunakan untuk mengolah dan menganalisis data secara efisien.

  • Struktur data utama: Series dan DataFrame.

  • Sangat penting untuk data science, machine learning, statistik, dan analisis data.

üëâ Series = satu kolom data (mirip list/array satu dimensi).
üëâ DataFrame = tabel data dua dimensi, mirip spreadsheet.

üì• 2. Instalasi & Setup

2.1 Contoh perintah di terminal:

pip install pandas

2.2 Lalu di python:

import pandas as pd

üß© 3. Membaca / Mengimpor Data

3.1 Contoh file CSV atau Excel:

df = pd.read_csv("data.csv")
df_excel = pd.read_excel("data.xlsx")

üìå Ini adalah langkah paling awal dalam workflow analisis data: membaca dataset.

üîç 4. Eksplorasi Data (EDA Basic)

üìå Menampilkan kolom dan ukuran data 

df.head()       # lihat 5 baris pertama
df.tail()       # lihat baris terakhir
df.shape        # ukuran (baris, kolom)
df.columns      # daftar nama kolom

üìä 5. Seleksi dan Filtering Kolom / Baris

üîπ Filter dengan kondisi:

df_filtered = df[df["Age"] > 30]

üîπ Seleksi kolom tertentu:

df_names = df[["Name", "Salary"]]

üìå Intinya: Pandas memungkinkan kita memilih subset data berdasarkan kondisi logis.

üõ† 6. Operasi Dasar pada DataFrame

‚ú® Mengurutkan Data

df.sort_values("Salary", ascending=False)

‚ú® Mengisi nilai hilang

df.fillna(0)

‚ú® Menghapus nilai hilang

df.dropna()

üìç Kamu belajar bagaimana membersihkan dataset sebelum analisis lebih lanjut.

üìà 7. Agregasi & Grouping Data

Contoh: Total gaji berdasarkan jabatan 

df.groupby("Job")["Salary"].sum()

üëâ Ini sangat berguna untuk menganalisis statistik per kelompok (statistik deskriptif).

üßÆ 8. Operasi Statistik Dasar

df.mean()       # rata-rata
df.median()     # median
df.describe()   # ringkasan statistik (count, mean, std, dll.)

üìå describe() sering dipakai untuk cepat mengetahui gambaran umum dataset.

üßæ 9. Menyimpan / Mengekspor Data 

df.to_csv("hasil.csv")
df.to_excel("hasil.xlsx")

üìå Ini penting ketika kamu sudah selesai membersihkan atau memproses data dan ingin menyimpannya.

üß™ Contoh Implementasi Praktik Pandas (Ringkas)

Misalkan kamu punya file employees.csv: 

Name,Age,Job,Salary
Alice,30,Engineer,70000
Bob,25,Designer,65000
Charlie,35,Engineer,80000

üëâ Contoh read, filter, dan group:

import pandas as pd

# Load data
df = pd.read_csv("employees.csv")

# EDA dasar
print(df.head())
print(df.describe())

# Filter: usia > 30
older_than_30 = df[df["Age"] > 30]
print(older_than_30)

# Group by pekerjaan ‚Üí rata-rata gaji
avg_salary_by_job = df.groupby("Job")["Salary"].mean()
print(avg_salary_by_job)
